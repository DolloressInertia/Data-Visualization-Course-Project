# Interactive Data Visualization Web Application

## Description
This project is a full-stack web application that allows users to upload datasets (CSV/XLSX) and automatically perform data analysis, visualization, and insight generation.

## Features
- Upload dataset
- Preview first 20 rows
- Automatic data cleaning
- Statistical analysis
- Interactive charts (Plotly)
- AI-like insights generation

## Technologies
Frontend: HTML, CSS, JavaScript
Backend: Python (FastAPI)
Libraries: pandas, numpy, plotly

## How to Run
1. Run all cells in Google Colab
2. Upload dataset
3. Click "Analyze Dataset"
4. View results
